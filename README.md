Task 3: SQL for Data Analysis

Objective

To analyze sales data using SQL and extract meaningful insights.

Dataset Used

Sales Data Sample from Kaggle.

Steps Performed

• Displayed sample records using SELECT and LIMIT

• Sorted orders by sales using ORDER BY

• Identified orders above average sales using subqueries

• Calculated total and average sales using GROUP BY, SUM, and AVG

• Performed INNER JOIN to combine sales and customer data

• Used LEFT JOIN to include all sales records

• Filtered high-value customers using HAVING clause

• Created indexes to optimize query performance

• Created a view to analyze high-value orders

Tools Used

• SQLite
• DB Browser for SQLite

Output

• SQL queries file (task3_sql_queries.sql)
• Screenshot PDF of query outputs
